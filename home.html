Important FAQ?

what is the Document Object Model (DOM)?

The DOM is essential for modern web development because it enables developers to create dynamic, interactive websites by interacting with the structure and style of web pages through JavaScript.

How do you select an element from the DOM?

we can select elements from the DOM using various JavaScript methods by using getElementByID(), getElementsByClassName(), getElementsByTagName(), querySelector() and so on.

What is the event delegation in the context of DOM, And is it useful?

Yes it is useful, event delegation optimizes our code, improves performance, and makes your JavaScript more scalable, particularly when working with large or dynamic DOM structures.

How do you manipulate an elements atribute's and stylr by using the DOM?

We can manipulate an elements attributes and styles using the DOM in JavaScript through various properties and methods. If we want to manipulate by using attributes we can use getAttribute(), setAttribute(), removeAttribute() etc. IF we want to manipulate by using style we can use element.style.property, Using classList for Styles etc.
